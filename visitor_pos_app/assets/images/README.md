# Assets Folder

## Logo Image

Please place your logo file here with the name `logo.png`.

**Requirements:**
- Filename: `logo.png`
- Recommended size: 512x512 pixels or larger
- Format: PNG with transparent background
- Aspect ratio: Square or landscape

The logo will be used in:
- Login screen (centered with animation)
- App bar header
- Dashboard header

If you don't have a logo yet, you can create a placeholder or use a company name text temporarily.
